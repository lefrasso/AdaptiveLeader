# Changelog

All notable changes to The Adaptive Leader are documented here. Releases use
[Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-07-19

### Added

- Complete 22-chapter leadership guide across all 10 supported languages.
- Full translations of the expanded team chapters and growth-mindset chapter.
- Chapter quizzes, practice reflections, spaced review, and growth dashboard.
- Four assessment tools, four themes, privacy controls, and static deployment.
- In-app version display and automated SemVer validation.
